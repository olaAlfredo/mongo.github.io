Inisio de sesion.

Inciar sesion mediante usuario y contraseña.

Una vez el usuario inicia sesion regresara un mensaje de inicio exitoso.

AUTORIZACIÓN REQUERIDA: ApiKey

Permite iniciar n veces sesion.

Authorization type: bearer token
Bearer format: sk_test_xxxxxxxxxxx

REQUEST BODY SCHEMA: application/json

username string required
Representa el nombre de usuario.
password string required
Representa la contraseña del usuario.

Responses

201 Nuevo Usuario creado
403 Error al crear usuario
